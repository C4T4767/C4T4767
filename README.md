## Featured Projects

### [Peekle](https://github.com/Peek-a-chu/Peekle)

- `frontend / backend / ai-server / extension` 으로 나뉜 **monorepo 구조**
- **Frontend**: Next.js 15, TypeScript, Tailwind CSS, Shadcn/UI
- **Backend**: Java 21 기반 서버, Docker 환경, Redis / MySQL 연동
- **AI Server**: Python 기반 처리 서버
- **Extension**: 백준 제출 결과를 감지하고, 풀이 정보와 코드를 수집하는 Chrome Extension
